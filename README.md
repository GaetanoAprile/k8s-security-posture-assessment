# k8s-security-posture-assessment

# Valutazione di Posture di Sicurezza Kubernetes (Minikube)

## Obiettivo del Progetto
Eseguire un assessment di un cluster Kubernetes locale (Minikube), identificare misconfigurazioni intenzionali tramite benchmark, applicare le correzioni (hardening) e misurare i miglioramenti.

## Tecnologie Utilizzate
* Minikube / k3s
* kube-bench
* kubeaudit
* Trivy

## Fasi del Progetto
- [x] **Fase 1:** Setup cluster e workload demo (vulnerabile)
- [ ] **Fase 2:** Assessment iniziale con tool di auditing
- [ ] **Fase 3:** Implementazione fix principali (RBAC, Pod Security)
- [ ] **Fase 4:** Re-assessment e metriche di confronto
- [ ] **Fase 5:** Documentazione e Golden Manifests
